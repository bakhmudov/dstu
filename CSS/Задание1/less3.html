По умолчанию браузер визуально отделяет абзацы друг от друга вертикальным отступом, что эквивалентноустановке свойств margin-top и margin-bottom в некоторое отличное от нуля значение.

В данном примере этим свойствам задано небольшое значение (0.2 em, т.е. 20% от высоты символа М).

Кроме того каждый абзац имеет отступ первой строки, который задан CSS-свойством text-indent.

Буквица - еще одни эффект, применяемый для визуального выделения начала обзаца. В CSS он может быть достигнут путем применения специфического стиля к первой букве абзаца.

Первой букве абзаца соответствует слектор вседоэлемента p:first-letter. Для этого всевдоэлемента в данном примере назначена высота 2em и красный полужирный шрифт.

Элемент strong по умолчанию делает вложенный текст полужирным, em - курсивом, но при помощи стилей такое поведение легко переопределить. Элемент span по умолчанию никак не влияет на вложенный в него текст - к нему всегда нужно применять стиль.
